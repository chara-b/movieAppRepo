---------ΒΗΜΑΤΑ ΕΚΤΕΛΕΣΗΣ WEB APP-----------

1.	Extract τον φάκελο WEB APP
2.	Δεν στέλνω τα movies.dat και ratings.dat αρχεία γιατί αυξάνουν τον όγκο της εργασίας και δεν θα ανεβαίνει στο eclass, παρόλα αυτά πρέπει να μπουν μέσα σε αυτόν τον φάκελο WEB APP για να διαβαστούν
3.	Μέσα από το node cli γράφουμε --> cd path_of_WebApp_Folder
4.	Πάλι μέσα από το node cli γράφουμε --> npm install (πάει και διαβάζει όλα τα modules που έχω εγκαταστήσει από το package.json file, τα βάζει εκεί ως dependencies... θα δημιουργήσει έναν φάκελο node_modules μέσα στον φάκελο WEB APP) 
5.	Εγώ ανοίγω τη βάση στον XAMPP συγκεκριμένα
6.	Μετά ξανά μέσα από το node cli γράφουμε --> node loadDB.js (κάνει 10 λεπτά να φορτώσει όλες τις ταινίες και να φτιάξει γενικά τη βάση με many2many relationship για τα genres με τις ταινίες. Αν τρέξει για 2η φορά αυτό το αρχείο η βάση δεν πειράζεται!)
7.	Επόμενη εντολή στο node cli είναι η --> node loadDB1.js (αυτό το αρχείο απλά φορτώνει τα ratings. Έχω απορία εδώ θα σας ρωτήσω αύριο την αναφέρω και στα σχόλια στον φάκελο που ανέβασα την εργασία στο eclass)
8.	Τελευταία εντολή για το node cli είναι η --> node server.js (έχει τα endpoints του api)
9.	Το frontend πρέπει να το σερβίρει κάποιος server αλλιώς δεν τρέχουν τα animations και τα
	modals του bootstrap και αυτό φέρει ως αποτέλεσμα να μην λειτουργεί το ui με το πάτημα 
	των buttons... Εγώ εγκατέστησα στα γρήγορα το "live server" ένα super lightweight extension
	στο vs code και βγάζει ένα "Go Live" κάτω δεξιά το πατάω και με πετάει στο chrome και μου
	τρέχει το frontend
